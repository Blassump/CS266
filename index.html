Calendar in C Programming

โปรแกรมปฏิทินเพื่อจัดการตารางเวลาในชีวิตประจำวัน


Let's start the program


Function 1 : Find out the Day (ค้นหาวัน)

- Do you want to continue? (Press Y or N)
   ต้องการใช้งานฟังก์ชันนี้ต่อหรือไม่ Y = ใช้ N = ไม่ใช้

- Enter date (DD MM YYYY) (กรอกวันที่เป็นตัวเลข) วัน 1 - 31 เดือน 1 - 12 ปี ค.ศ
   แสดงวันที่ พร้อมเวลาที่ค้นหาบนเครื่องคอมพิวเตอร์

- Press any key to continue .. กดปุ่มใดบนคีย์บอร์ดก็ได้เพื่อจบการทำงาน

Function 2 : Print all the day of month

(แสดงปฏิทินของแต่ละเดือน)

- Do you want to continue? (Press Y or N)
   ต้องการใช้งานฟังก์ชันนี้ต่อหรือไม่ Y = ใช้ N = ไม่ใช้

- Enter month and year (MM YYYY) ระบุเดือน 1 – 12 ปี ค.ศ
   การใช้งานปุ่ม p n s q ในฟังก์ชันที่ 2
   p = กดเพื่อดูเดือนก่อนหน้า n = กดเพื่อดูเดือนถัดไป
   s = กดเพื่อดูโน้ตที่ได้จดไว้ q = จบการทำงานของฟังก์ชันนี้



Function 3 : Add note (เพิ่มโน้ต)

- Enter the date (DD MM YYYY)
   ระบุวันที่ที่ต้องการโน้ต วัน 1 – 31 เดือน 1 – 12 ปี ค.ศ

- Do you want to continue? (Press Y or N)
   ต้องการใช้งานฟังก์ชันนี้ต่อหรือไม่ Y = ใช้ N = ไม่ใช้

- Enter the note (ระบุโน้ตที่ต้องการพิมพ์)

- Press any key กดปุ่มไหนก็ได้เพื่อจบการทำงานของฟังก์ชัน



Function 4 : EXIT (จบการทำงานของโปรแกรม)

- Do you want to continue? (Press Y or N)
   ต้องการใช้งานฟังก์ชันนี้ต่อหรือไม่ Y = ใช้ N = ไม่ใช้
